# THE BUMPER CROP

*a Backrooms · Level 10 story — **created by kamsamnor***

A first-person exploration game set in Backrooms Level 10, "The Bumper Crop":
an endless expanse of wheat and barley under a sky that never changes…
as long as you stay close to home.

## The rules of the level

- **The house is safe.** Cook, read, rest. The therapist — a shadow person
  named Dr. Umbra — sits in the living room and will talk with you about
  anything. He remembers your dreams, your blackouts, and how far you've walked.
- **The fields are not.** The further you go, the darker the world follows.
  Deep enough out, tall dark things start walking the wheat. Don't stare at
  them. Don't stand still. Walk home — daylight un-happens them.
- **Sleep to dream.** Dreams are adventures: floating wheat islands, neon
  voids, a stalk as tall as weather. You always wake up with something.
- **Almond water** is hidden in the far fields. Drink it when the edges of
  your eyes go dark.
- You cannot die. If the dark catches up with you, the fields carry you home.

## How to play

Open the game link in Chrome on your phone, then **⋮ → Add to Home Screen**
to install it like a real app.

| | Phone | Computer |
|---|---|---|
| Move | left-thumb joystick | WASD |
| Look | drag the right side | mouse (click once to grab) |
| Interact | tap | E or click |
| Drink almond water | tap the bottle button | Q |

## Tech (for grown-ups)

No build step, no frameworks, no accounts. Plain HTML + ES modules +
[three.js](https://threejs.org) (vendored in `js/vendor/`). Every texture,
model and sound is generated by code at runtime — there isn't a single
art file in the repo. Works offline after the first visit (service worker).

Run locally: `python3 -m http.server` in this folder, open
`http://localhost:8000`.

Deploy: enable GitHub Pages (Settings → Pages → deploy from branch), done.

The therapist's brain is rule-based for now (`js/therapist.js`,
`RuleBrain`) but sits behind an async interface so a real Claude-powered
brain can drop in later without changing anything else.
